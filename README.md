# Multi-laguage implementations

- how these languages differ
- practice programming
- make it easier for beginners to understand lower level languages

Write these programs in all languages showed here:
- C
- C++
- Fortran
- Java
- Haskell
- Rust
- Nasm x86_64 Linux
- Fasm x86_64 Linux
